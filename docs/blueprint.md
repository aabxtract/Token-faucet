# **App Name**: Web3 Token Faucet

## Core Features:

- Connect Wallet: Allow users to connect their Ethereum wallet using RainbowKit.
- Display User Info: Show the connected user's address and token balance.
- Claim Tokens: Enable users to claim tokens with a single click using Wagmi.
- Smart Contract Interaction: Interact with the Faucet smart contract to send tokens.
- Prevent Double Claiming: Utilize a smart contract modifier to prevent users from claiming tokens multiple times.
- Token and Faucet Deployment: Deploy ERC20 and faucet tokens to a local hardhat network
- Display Claim Status: Provide a success message or error feedback after claiming.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) for a calming, trustworthy feel.
- Background color: Off-white (#F9F9F9) to provide a minimal and clean look.
- Accent color: Light green (#90EE90) to highlight CTAs and success messages.
- Body and headline font: 'Inter', a sans-serif, for a modern, clean, and readable design.
- Single centered card layout for a focused and user-friendly experience.
- Smooth transitions and subtle animations to enhance the user experience during wallet connection and token claiming.
- Simple, outlined icons to represent blockchain and wallet interactions.
